# European-Tech-Layoffs-Analysis
Projeto final do curso de Análise de Dados da Coursera. Análise de layoffs no setor tech europeu (2020-2025) usando R, SQL e Tableau. Inclui pipeline completo: limpeza de dados, exploração por país/indústria e dashboard interativo. Demonstra competências em tratamento de dados, consultas complexas e visualização de insights.
